Cardiovascular Disease (CVD) Prevalence Factor Analysis Project Summary
Here is a simplified English translation of your project report overview.

Project Overview:
This project report summarizes the statistical analysis of key factors affecting the rate of Cardiovascular Disease (CVD) across different communities in the UK, using data from the Office for National Statistics (ONS).

Project Goal:
To find out and measure which local factors (obesity, smoking, poverty, and wellbeing) have the biggest influence on the percentage of people with CVD in different UK local authority regions.
